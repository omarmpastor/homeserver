# Uso de scripts de configuración y puesta en marcha

## Puesta en marcha

🧪 Validación segura
```bash
stackctl dry-run
```

🔁 Pull de imagenes
```bash
stackctl pull
```

🚀 Deploy completo
```bash
stackctl up
```

⛔ Stop completo
```bash
stackctl down
```

💣 Borrado de datos
```bash
stackctl purge-volumes
```

📊 Estado
```bash
stackctl status
```

## Backups

💾 Activar backups
```bash
backupsctl up
```

♻️ Recovery snapshots
```bash
backupsctl recovery
```
